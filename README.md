# Banners SmartyAds

Banners HTML5 para campanas de programatica + IA en SmartyAds DSP.

Brand: rodriguezsamblas.com (BR1+ MERGE)

## Formatos

- `banner-300x600.html` - Half Page
- `banner-970x250.html` - Billboard

## Uso en SmartyAds (3-party tag)

Pega este codigo en el campo "3-party tag":

```html
<iframe src="https://banners-smartyads.vercel.app/banner-300x600.html" width="300" height="600" frameborder="0" scrolling="no" style="border:none;overflow:hidden;"></iframe>
```

Reemplaza el formato segun el banner.
